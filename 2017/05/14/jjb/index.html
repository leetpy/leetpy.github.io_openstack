jenkins job builder

May 14, 2017 | ci | Hits

使用 jenkins 时，可以通过在 web 上创建 job，但这种方式在少量 job 时还可以，如果涉及到大量 job, 一是难于管理， 二是代码不能复用。通常的做法是，使用代码来编写 jenkins job， 并使用 git 来管理。在 jenkins 上创建一个 seed job， 然后当我们的代码有提交时，自动更新 job。

目前有两种方式用代码来创建 jenkins job， 一种是使用 job-dsl 插件，通过 groovy 脚本来编写，另一种就是 openstack 社区现在推荐的：使用 jenkins-job-builder ＋ yaml 来编写。虽然 groovy 也很方便，但是在插件转换成 xml 时，严重依赖与 job-dsl的版本。

eventlet 使用

Apr 19, 2017 | python | Hits

熟悉openstack 的朋友可能经常见到eventlet这个库。eventlet是一个python 并发网络库，eventlet采用协程实现，具有如下优点：

  • 使用epoll, kqueue，libevent 提供高性能无阻塞I/O
  • 采用协程，无阻塞I/O，并发高，消耗资源少
  • 事件分发

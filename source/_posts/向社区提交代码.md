# 提交代码

## git review安装

```python
pip install git-review
```

## git 配置

```shell
# 这里的名字必须是gerrit的名字
git config gitreview.username lee

# ssh用不了，使用https方式提交
git config gitreview.scheme https
git config gitreview.port 443

git review -s -v
```

## 提交代码

```shell
git add --all
git commit -m "commit msg"

git review
```
